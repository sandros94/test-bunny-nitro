# Demo

This is a demo repository showcasing a custom Nitro preset for Bunny.net's Edge Scripting platform. The preset enables developers to create serverless applications that run on Bunny.net's edge network.

https://test-nitro-preset-un20f.bunny.run

## Features

- Custom Nitro preset for Bunny.net Edge Scripting
- SSR support with Preact (with streaming)
- WebSocket support using `crossws`
- Static file serving via inlined VFS
- Local development running on Deno
